grumpy_checks.lint

module documentation
(source)

Check lint. This module defines and registers the various lint checking functions. By design all checks are registered by default and must be explicilty unregistered (grumpy.lint.unregister(name: str)) if desired to be skipped. This module contains the following functions: * has_flake8 - ensures that a .flake8 file exists * run_flake8 - run the flake8 linter and capture the output The LINT_CHECKS Callable object is a CheckCollection which contains the register of the current lint checks, see grumpy.lint.LINT_CHECKS.collection New functions that return a CheckResponse object can be added to the register with LINT_CHECKS.register(func: Callable[[Any], CheckResponse])
