Check Collection Group. A class that defines an object whose primary function is to allow grouping of one of more CheckCollections (groups of check functions) into a topic. Objects of class CheckCollectionGroup are themselves Callable, creating and displaying a table as individual checks run. Methods ======= register: function closure (decorator) to add an additional Callable to the collection unregister: remove Callable from collection by name get_registered: return a list of the names of registered Callables Attributes ========== @ivar collection: dictionary str -> Callable containing functions registered in the collection @ivar issues: int, increments whenever a check function fails
