def as_CheckResponse(func): (source)

Create a CheckResponse. Wrap the Tuple[bool, str] result of a callable with a CheckResponse type output. This is convenient for allowing the function name at definition to be used as the name for the CheckResponse object. @param func: A Callable object that takes any arguments and returns a Tuple[bool, str] @returns: A Callable, original function definition with wrapped return value.
